# Agent Guidelines

## Code Quality
- Write clean, readable, and maintainable code
- Follow established project conventions
- Document complex logic appropriately

## Task Execution
- Break down complex tasks into manageable steps
- Verify results before claiming completion
- Handle errors gracefully with informative messages

## Tool Usage
- Use tools appropriately for the task at hand
- Understand tool capabilities and limitations
- Chain tools efficiently when needed

## Verification
- Test changes before marking tasks complete
- Verify user experience, not just code correctness
- Consider edge cases and error conditions
